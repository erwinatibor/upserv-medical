# Images — Lumina Dental

Place all site images in this folder. Below are the required specs:

| Filename               | Size         | Usage                                  |
|-----------------------|--------------|----------------------------------------|
| `og-image.jpg`         | 1200x630     | Social sharing preview                 |
| `favicon.ico`          | 32x32        | Browser tab                            |
| `apple-touch-icon.png` | 180x180      | iOS home screen                        |
| `studio-hero.jpg`      | 1920x1080    | Optional hero background               |
| `studio-interior.jpg`  | 900x700      | About section frame                    |
| `dr-voss.jpg`          | 400x400      | Dr. Alexandra Voss portrait            |
| `dr-kim.jpg`           | 400x400      | Dr. Rafael Kim portrait                |
| `dr-sato.jpg`          | 400x400      | Dr. Naomi Sato portrait                |
| `dr-levy.jpg`          | 400x400      | Dr. Marcus Levy portrait               |

## Photography Style Guide

- Low-key studio lighting
- Dark or neutral backgrounds
- Professional attire
- Avoid bright clinical whites — they conflict with the dark design aesthetic

## Naming Convention

- Use lowercase, hyphenated names: `dr-voss.jpg`
- Use `.jpg` for photos, `.png` for graphics with transparency
- Use `.svg` for icons and logos where possible
